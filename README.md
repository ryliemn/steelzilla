# Steelzilla

Portfolio site for Steelzilla — CNC plasma cut, hand-finished metal artwork.

Built with [Astro](https://astro.build) and deployed as a fully static site.

## Commands

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server at `http://localhost:4321` |
| `npm run build` | Build static site to `dist/` |
| `npm run preview` | Preview the production build locally |
| `npm run lint` | Lint with oxlint |
| `npm run fmt` | Format with oxfmt |
| `npm run fmt:check` | Check formatting without writing |

## Stack

- [Astro](https://astro.build) — static site framework
- [React](https://react.dev) — interactive islands (navbar, gallery, contact form)
- [shadcn/ui](https://ui.shadcn.com) — UI components (button, navigation menu, sheet)
- [Tailwind CSS](https://tailwindcss.com) — styling
- [Formspree](https://formspree.io) — contact form backend
- [Cloudflare Images](https://developers.cloudflare.com/images/) — image CDN
